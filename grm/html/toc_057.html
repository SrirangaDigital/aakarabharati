ಶ್ರೀ ವಿಷ್ಣುಧರ್ಮೋತ್ತರ ಮಹಾಪುರಾಣಂ – ತೃತೀಯಖಂಡೇ – ಭಾಗ – ೨
ಆರಂಭಿಕ ಪುಟ
ವಿಷಯಾನುಕ್ರಮಣಿಕೆ
  • ತೊಂಭತ್ತಾರನೆಯ ಅಧ್ಯಾಯ
    • ದೇವತೆಗಳ ವಿಗ್ರಹಪ್ರತಿಷ್ಠೆಗೆ ತಕ್ಕ ದೇಶಕಾಲನಿರೂಪಣ
  • ತೊಭತ್ತೇಳನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಷ್ಠಾಕಲ್ಪಕ್ಕೆ ಸೇರಿದ ದೀಕ್ಷಾವಿಧಿನಿರೂಪಣ.
  • ತೊಂಭತ್ತೆಂಟನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಷ್ಠಾಕಲ್ಪಕ್ಕೆ ಸೇರಿದ ತೋರಣವರ್ಣನ.
  • ತೊಂಭತ್ತೊಂಭತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಷ್ಠಾಕಲ್ಪದ ಪಂಚಗವ್ಯವರ್ಣನ.
  • ನುರನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಮಾಶುದ್ಧಿಗೆ ಉಪಯುಕ್ತವಾದ ಅರ್ಚಾಶೌಚಕಥನ.
  • ನೂರ ಒಂದನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಮೆಗಳ ಅಧಿವಾಸನಾವರ್ಣನ.
  • ನೂರ ಎರಡನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರಇಮೆಗಳಲ್ಲಿ ದೇವತೆಗಳ ಆವಾಹನವರ್ಣನ.
  • ನೂರ ಮೂರನೆಯ ಅಧ್ಯಾಯ
    • ಎಲ್ಲಾ ದೇವತೆಗಳಿಗೂ ಪ್ರತಿಮೆಗಳಲ್ಲಿ ಆವಾಹನಕ್ರಮ.
  • ನೂರ ನಾಲ್ಕನೆಯ ಅಧ್ಯಾಯ
    • ಬ್ರಹ್ಮನೇ ಮೊದಲಾದ ದೇವತೆಗಳ ಆಹ್ವಾನ ನಿರೂಪಣ.
  • ನೂರ ಐದನೆಯ ಅಧ್ಯಾಯ
    • ಲೋಕಾದ್ಯಾವಾಹನವರ್ನನ.
  • ನೂರ ಆರನೆಯ ಅಧ್ಯಾಯ
    • ಇತರ ಎಲ್ಲಾ ದೇವತೆಗಳ ಆವಾಹನವನವರ್ಣನ.
  • ನೂರ ಏಳನೆಯ ಅಧ್ಯಾಯ
    • ವಾಸುದೇವನ ವಿಬೋಧನವರ್ನನ.
  • ನೂರ ಎಂಟನೆಯ ಅಧ್ಯಾಯ
    • ಸರ್ವವ್ಯಾಪಕನಾದ ವಿಷ್ಣುವಿಗೆ ಪ್ರತಿಮೆಯಲ್ಲಿ ಆವಾಹನವೇತಕ್ಕೆ ಎಬ ಪ್ರಶ್ನೆಗೆ ಉತ್ತರವಾಗಿ ಅವನ ಪರಾಪರಗಳೆಂಬ ರೂಪ ಭೇದಗಳ ನಿರೂಪಣಪೂರ್ವಕ ಆವಾಹನ ಪ್ರಯೋಜನವರ್ಣನ.
  • ನೂರ ಒಂಭತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಷ್ಠಾಕಲ್ಪದ ಹೋಮವಿಧಿನಿರೂಪಣ.
  • ನೂರ ಹತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಷ್ಠಾಕಲ್ಪದ ವಿಬೋಧನ ಮಂತ್ರಗಳನಿರೂಪಣ.
  • ನೂರ ಹನ್ನೊಂದನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರತಿಷ್ಠೆತನಾದ ಭಗವಂತನ ಬೃಹತ್ಸ್ನಪನವರ್ಣನ, ಮತ್ತು ಪ್ರಶಂಸಾ.
  • ನೂರ ಹನ್ನೆರಡನೆಯ ಅಧ್ಯಾಯ
    • ಭೋಗದಾನವೆಂಬ ಸಮಂತ್ರಕಪೂಜಾವಿಧಾನ.
  • ನೂರ ಹದಿಮೂರನೆಯ ಅಧ್ಯಾಯ
    • ಮಧುಪರ್ಕ ನಿವೇದನಕ್ರಮ.
  • ನೂರ ಹದಿನಾಲ್ಕನೆಯ ಅಧ್ಯಾಯ
    • ಸಮಂತ್ರಕವಾದ ಇಜ್ಯಾವರ್ಣನ.
  • ನೂರ ಹದಿನೈದನೆಯ ಅಧ್ಯಾಯ
    • ಶಂಕರಗೀತೋಕ್ತ ವಿಧಾನದಿಂದ ಸಾತ್ವತನ ಇಜ್ಯಾವರ್ನನ.
  • ನೂರ ಹದಿನಾರನೆಯ ಅಧ್ಯಾಯ
    • ವೈದಿಕೇಜ್ಯಾವಿಧಾನದಿಂದಲೂ. ಸಾತ್ತ್ವಿಕೇಜ್ಯಾವಿಧಾನದಿಂದಲೂ ಪೂಜೆಯು ನಡೆದನಂತರ ತೋರಣೋದ್ಧಾರವಿಧಿವರ್ನನ.
  • ನೂರ ಹದಿನೇಳನೆಯ ಅಧ್ಯಾಯ
    • ಭಗವಂತನ ತ್ರಾವಿಧಿ (ಉತ್ಸವ) ವರ್ಣನ.
  • ನೂರ ಹದಿನೆಂಟನೆಯ ಅಧ್ಯಾಯ
    • ಮಹಾವಿಷ್ಣುವಿನ ಶರೀರಭೂತರಾದ ದೇವತೆಗಳಲ್ಲಿ ಅವನವನಿಗೆ ಇಷ್ಟವಾದ ಫಲಕ್ಕೆ ತಕ್ಕಂತೆ ನಡೆಸಬಹುದಾದ ತತ್ತತ್ಕಾಮ ಪೂಜನ ವರ್ಣನ.
  • ನೂರ ಹತ್ತೊಭತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಕಾರ್ಯಾರಂಭದಲ್ಲಿ ಕಾರ್ಯಸಿದ್ಧಿಗೆ ತಕ್ಕಂತೆ ಪೂಝಿಸಬೇಕಾದ ದೇವತಾ ವಿಶೇಷನಿರ್ಣಯ.
  • ನೂರ ಇಪ್ಪತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರಾದುರ್ಭಾವರೂಪಗಳಾದ ಭಗವಂತನ ಮೂರ್ತಿಗಳನ್ನು ಪೂಜಿಸಲು ವಿಹಿತವಾದ ಕಾಲನಿಯಮ ನಿರೂಪಣ.
  • ನೂರ ಇಪ್ಪತ್ತೊಂದನೆಯ ಅಧ್ಯಾಯ
    • ಭಗವಂತನಪ್ರಾದುರ್ಭಾವರೂಪಗಳಾದ ಮೂರ್ತಿಗಳನ್ನು ಪೂಜಿಸಲು ವಿಹಿತವಾದ ದೇಶಗಳ ನಿರೂಪಣ.
  • ನೂರ ಇಪ್ಪತ್ತೆರಡನೆಯ ಅಧ್ಯಾಯ
    • ಕಾಮ್ಯಫಲಕ್ಕಾಗಿ ಸರ್ವಶರೀರಿಯಾದ ವಾಸುದೇವನನ್ನು ಪೂಜಿಸುವವನು ಕಾರ್ಯಸಿದ್ಧಿಗಾಗಿ ಜಪಿಸಬೇಕಾದ ಭಗವಂತನ ನಾಮ ವಿಶೇಷಗಳ ನಿರ್ಣಯ.
  • ನೂರ ಇಪ್ಪತ್ತಮೂರನೆಯ ಅಧ್ಯಾಯ
    • ಕಾರ್ಯಾರಂಭಕಾಲದಲ್ಲಿ ಜಪಿಸಬೇಕಾದ ಭಗವಂತನ ನಾಮವಿಶೇಷಗಳ ನಿರ್ಣಯ.
  • ನೂರ ಇಪ್ಪತ್ತನಾಲ್ಕನೆಯ ಅಧ್ಯಾಯ
    • ವಾಸುದೇವಪೂಜಾನಿಷ್ಠನಾದ ಮನುಷ್ಯನಿಗೆ ಆಯಾಯ ಕಾಲದಲ್ಲಿ ಉಚ್ಚರಿಸಬೇಕಾದ ಭಗವನ್ನಾಮ ವಿಶೇಷಗಳ ನಿರೂಪಣ.
  • ನೂರ ಇಪ್ಪತ್ತೈದನೆಯ ಅಧ್ಯಾಯ
    • ಆಯಾಯಕ್ಷೇತ್ರಗಳಲ್ಲಿ ಭಕ್ತನಾದವನು ಕೀರ್ತನ ಮಾಡಬೇಕಾದ ಭಗವನ್ನಾಮ ವಿಶೇಷಗಳ ನಿರೂಪಣ.
  • ನೂರ ಇಪ್ಪತ್ತಾರನೆಯ ಅಧ್ಯಾಯ
    • ಇಷ್ಟ ದೇವತೆಯ ಸಾಮಾನ್ಯವಾದ ಪೂಜೆ ಮತ್ತು ಅಭಿಮತ ಫಲಕ್ಕೆ ತಕ್ಕ ವಿಶೇಷವಾದ ಫೂಜೆ ಮತ್ತು ಉಪವಾಸ ಫಲ ಇವುಗಳ ನಿರೂಪಣ.
  • ನೂರ ಇಪ್ಪತ್ತೇಳನೆಯ ಅಧ್ಯಾಯ
    • ತ್ರಿಮೂರ್ತಿರೂಪ ನಿರೂಪಣ.
  • ನೂರ ಇಪ್ಪತ್ತೆಂಟನೆಯ ಅಧ್ಯಾಯ
    • ಪೌರುಷೀಪ್ರತಿಪತ್ತೆಂಬ ವೈಷ್ಣವವ್ರತ ನಿರೂಪಣ.
  • ನೂರ ಇಪ್ಪತ್ತೊಂಭತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಪ್ರಕೃತಿಪುರುಷರ ಸಂಘಾತರೂಪರಾದ ಲಕ್ಷ್ಮೀ ಮತ್ತು ವಾಸುದೇವ ಇವರ ಪೂಜಾ ವಿಧಾನ.
  • ನೂರ ಮೂವತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ನಾಸತ್ಯ ಪೂಜಾರೂಪವಾದ ಚಕ್ಷುರ್ವ್ರತವರ್ಣನ.
  • ನೂರ ಮೂವತ್ತೊಂದನೆಯ ಅಧ್ಯಾಯ
    • ಬಾಲೆಂದು ದ್ವಿತೀಯಾವ್ರತವರ್ಣನ.
  • ನೂರ ಮೂವತ್ತೆರಡನೆಯ ಅಧ್ಯಾಯ
    • ಅಶೂನ್ಯಶಯನದ್ವಿತೀಯಾ ಎಂಬ ವೈಷ್ಣವವ್ರತವರ್ಣನ.
  • ನೂರ ಮೂವತ್ತಮೂರನೆಯ ಅಧ್ಯಾಯ
    • ತ್ರಿವಿಕ್ರಮವ್ರತವೆಂಬ ವೈಷ್ಣವವ್ರತವರ್ಣನ.
  • ನೂರ ಮೂವತ್ತನಾಲ್ಕನೆಯ ಅಧ್ಯಾಯ
    • ದ್ವಿತೀಯತ್ರೈವಿಕ್ರಮವ್ರತವರ್ನನ.
  • ನೂರ ಮೂವತ್ತೈದನೆಯ ಅಧ್ಯಾಯ
    • ತೃತೀಯತ್ರೈವಿಕ್ರಮವ್ರತವರ್ಣನ
  • ನೂರ ಮೂವತ್ತಾರನೆಯ ಅಧ್ಯಾಯ
    • ವಿಷ್ಣುತ್ರಿಮೂರ್ತಿ ವ್ರತವರ್ನನ.
  • ನೂರ ಮೂವತ್ತೇಳನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಮೂರ್ತಿವ್ರತವರ್ನನ.
  • ನೂರ ಮೂವತ್ತೆಂಟನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಮೂರ್ತಿವ್ರತವರ್ಣನ.
  • ನೂರ ಮೂವತ್ತೊಂಭತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಮೂರನೆಯ ಚತುರ್ಮೂರ್ತಿವ್ರತವೆಂಬ ವೈಸ್ಣವ್ರತ.
  • ನೂರ ನಲವತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ನಾಲ್ಕನೆಯ ಚತುರ್ಮೂರ್ತಿವ್ರತ.
  • ನೂರ ನಲವತ್ತೊಂದನೆಯ ಅಧ್ಯಾಯ
    • ವೇದವ್ರತವೆಂಬ ಐದನೆಯ ಚತುರ್ಮೂರ್ತಿವ್ರತ.
  • ನೂರ ನಲವತ್ತೆರಡನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಮೂರ್ತಿಕಲ್ಪದ ಆಶ್ರ,ಅವ್ರತವರ್ನನ.
  • ನೂರ ನಲವತ್ತಮೂರನೆಯ ಅಧ್ಯಾಯ
    • ಅಗ್ನಿ ವ್ರತವೆಂಬ ಏಳನೆಯ ಚತುರ್ಮೂರ್ತಿವ್ರತವರ್ಣನ.
  • ನೂರ ನಲವತ್ತನಾಲ್ಕನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಯುಗವ್ರತವೆಂಬ ಎಂಟನೆಯ ಚತುರ್ಮೂರ್ತಿವ್ರತ.
  • ನೂರ ನಲವತ್ತೈದನೆಯ ಅಧ್ಯಾಯ
    • ಸಾಗರವ್ರತವೆಂಬ ವಿಷ್ಣುಚತುರ್ಮೂರ್ತಿವ್ರತವರ್ಣನ.
  • ನುರ ನಲವತ್ತಾರನೆಯ ಅಧ್ಯಾಯ
    • ಧ್ವಜವ್ರತವೆಂಬ ಚತುರ್ಮೂರ್ತಿಕಲ್ಪದ ವ್ರತವಿಶೇಷವರ್ಣನ.
  • ನೂರ ನಲವತ್ತೇಳನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಮೂರ್ತಿಕಲ್ಪದ ದೇವಮೂರ್ತಿವ್ರತವರ್ಣನ
  • ನೂರ ನಲವತ್ತೆಂಟನೆಯ ಅಧ್ಯಾಯ
    • ಚತುಮೂರ್ತಿಕಲ್ಪದ ಆಯುಧವ್ರತವರ್ಣನ.
  • ನೂರ ನಲವತ್ತೊಂಭತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಮೂತಿವ್ರತಕಲ್ಪದ ಫಲಾಹಾರಹರಿಪ್ರಿಯವ್ರತವರ್ಣನ.
  • ನೂರ ಐವತ್ತನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಮೂತಿಕಲ್ಪದ ಆನಂತವ್ರತವರ್ಣನ.
  • ನೂರ ಐವತ್ತೊಂದನೆಯ ಅಧ್ಯಾಯ
    • ಚತುರ್ಮೂತಿಕಲ್ಪದ ಶ್ರೀ ವಿಷ್ಣುವ್ರತವರ್ಣನ.
  • ನೂರ ಐವತ್ತೆರಡನೆಯ ಅಧ್ಯಾಯ
    • ಪಂಚಮೂರ್ತಿಪಂಚಮಹಾಭೂತವ್ರತವರ್ಣನ.
  • ನೂರ ಐವತ್ತಮೂರನೆಯ ಅಧ್ಯಾಯ
    • ಪಂಚವ್ರತವಿಧಿಯ ಸಂವತ್ಸರ ಪೂಜೆಯಲ್ಲಿ ಅಶ್ವಮೇಧಫಲವರ್ಣನ.
  • ನೂರ ಐವತ್ತನಾಲ್ಕನೆಯ ಅಧ್ಯಾಯ
    • ಪಂಚಮೂರ್ತಿವಿಧಿಯ ಲಕ್ಷ್ಮೀವ್ರತವರ್ಣನ.
  • ನೂರ ಐವತ್ತೈದನೆಯ ಅಧ್ಯಾಯ
    • ಪಂಚಮೂರ್ತಿವಿಧಿಯ ಆಯುಧಾದಿವ್ರತವರ್ನನ
  • ನೂರ ಐವತ್ತಾರನೆಯ ಅಧ್ಯಾಯ
    • ಷಣ್ಮೂರ್ತಿವ್ರತವರ್ಣನ.
  • ನೂರ ಐವತ್ತೇಳನೆಯ ಅಧ್ಯಾಯ
    • ಸುಭಾಸ್ವರಾದಿ ಪಿತೃವ್ರತವರ್ಣನ.
  • ನೂರ ಐವತ್ತೆಂಟನೆಯ ಅಧ್ಯಾಯ
    • ಪಾತಾಲಗಳ ವ್ರತವರ್ನನ.
